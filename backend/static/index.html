StepIn API

Welcome to the StepIn API for physical meeting management!

API Documentation

The full API documentation is available at the following URLs:

Swagger UI: /docs

ReDoc: /redoc

API Endpoints

Users

Create a new user: POST /api/users

Get user details: GET /api/users/{email}

Meetings

Create a new meeting: POST /api/meetings

Get meeting details: GET /api/meetings/{meeting_id}

Get nearby meetings: POST /api/meetings/nearby

Join a meeting: POST /api/meetings/{meeting_id}/join

Leave a meeting: POST /api/meetings/{meeting_id}/leave

Get meeting participants: GET /api/meetings/{meeting_id}/participants

End a meeting: POST /api/meetings/{meeting_id}/end

Chat

Send a chat message: POST /api/chat/{meeting_id}

Get chat messages: GET /api/chat/{meeting_id}

Example Request

Creating a new user:

POST /api/users
Content-Type: application/json

{
    "email": "user@example.com",
    "name": "John Doe",
    "age": 30,
    "gender": "male"
}

Example Response

{
    "success": true
}
